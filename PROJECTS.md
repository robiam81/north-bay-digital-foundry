### 🔧 Potable Pipeline Engineering Suite

Status: Tool (`pipeline-calc.html`)

Seven preliminary-design calculators for pressurized potable water pipelines —
Hazen-Williams head loss, velocity check, restrained length, Barlow hoop
stress, Modified Iowa deflection, Joukowsky surge, and buoyancy/flotation.

Goals

- [x] Build calculator suite (pure engineering layer + UI layer)
- [x] Document method, assumptions, and sources inline per calculator
- [x] Integrate into website (project card)
- [ ] Real screenshot thumbnail on the project card

### 🔧 Storm Drainage Toolkit

Status: Tool (`storm-drainage-calc.html`)

Seven quick-reference calculators for small-site storm drain design —
Rational Method peak flow, pipe sizing, sump inlet capacity, detention basin
sizing, outlet structure (orifice + weir), culvert screening, and riprap
apron sizing.

Goals

- [x] Build calculator suite (pure engineering layer + UI layer)
- [x] Document method, assumptions, and sources inline per calculator
- [x] Integrate into website (project card)
- [ ] Verify hydraulic coefficients against HEC-22 / HDS-5 / HEC-14 (PE review)
